# Template

This is a minimal starter template repository for a **lim**-project.

## Getting started

You can get started with a lim-machine quickly:

1. Edit `src/index.ts` based on your preferences.
2. Open this directory with a terminal
3. Run `lim serve src/index.ts`

## Configuration

Machine specific configurations are automatically loaded from a `config.yml` file.

Secrets are loaded from `.env` automatically.

## Adding packages

You can install new packages with your favorite package manager.

`npm install gpio`
